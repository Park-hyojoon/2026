# 새로운 기능 요청 2가지 작업 계획서

## 1. 성경 본문 검색 및 삽입 기능

### 개요
`https://www.bskorea.or.kr/bible/korbibReadpage.php` 사이트를 이용하여 성경 장/절(예: "창 1:5")을 검색하고, 결과를 'Bible Body' 텍스트 박스에 자동으로 삽입하는 기능입니다.

### 구현 방안

#### A. UI 변경
-   **위치**: 'Bible Chapter/Verse (All Slides):' 라벨 아래에 검색 영역 추가.
-   **구성**:
    -   검색 입력창: "창 1:5" 형식 입력.
    -   [검색] 버튼: 클릭 시 웹 크롤링 시작.
    -   결과 반영: 검색된 성경 구절을 `Bible Body (Slide 5)` 영역에 줄바꿈과 함께 삽입.

#### B. 검색 로직 (백엔드)
-   `requests`와 `BeautifulSoup` 라이브러리를 사용하여 사이트의 HTML을 파싱합니다.
-   사용자가 입력한 약어(창, 출 등)를 사이트에서 인식하는 형식으로 변환하여 요청을 보냅니다.
-   검색 결과에서 본문 텍스트만 추출하여 GUI에 전달합니다.

## 2. 작업 완료 후 Chrome 자동 실행 및 이동

### 개요
PPT 생성 프로세스가 성공적으로 마무리되면, Google Chrome 브라우저를 실행하여 지정된 URL(`http://yeeun.synology.me:5000/`)로 즉시 이동합니다.

### 구현 방안
-   `os` 또는 `webbrowser` 모듈을 사용하여 시스템 명령을 실행합니다.
-   성공 메시지 박스가 띄워진 직후 또는 확인을 누른 후 브라우저가 열리도록 구성합니다.
