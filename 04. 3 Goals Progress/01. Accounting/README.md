# AI 회계 학습 도우미

로컬에서 작동하는 AI 기반 회계 학습 프로그램입니다.

## 주요 기능

### 1. GUI 기반 프로그램
- 터미널이 아닌 사용자 친화적인 GUI 인터페이스
- Tkinter 사용 (Python 기본 내장, 추가 설치 불필요)

### 2. PDF 학습 자료 분석
- OCR로 변환된 PDF 파일에서 텍스트 추출
- AI가 내용을 분석하여 학습 문제 자동 생성

### 3. 학습 기록 및 통계
- 모든 학습 세션 자동 저장
- 정답률, 학습 횟수 등 통계 제공
- 학습 진도 추적

### 4. 취약점 분석
- 자주 틀리는 주제 키워드 자동 분석
- 오답 노트 제공
- 맞춤 학습 추천

### 5. 로컬 실행
- 인터넷 연결만 있으면 어디서든 사용 가능
- Google Gemini API 사용 (무료)
- 개인 정보 보호 (로컬 저장)

## 설치 방법

### 1. Python 설치
Python 3.8 이상이 필요합니다.

### 2. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

### 3. Google Gemini API 키 받기
1. [Google AI Studio](https://aistudio.google.com/app/apikey) 방문
2. 무료 API 키 발급
3. API 키 복사

## 사용 방법

### 1. 프로그램 실행
```bash
cd quiz_program
python gui_app.py
```

### 2. 초기 설정
1. Google Gemini API 키 입력 (한 번만 입력하면 자동 저장됨)
2. 학습할 PDF 파일 선택
3. 생성할 문제 수 설정 (3~10개)

### 3. 학습 시작
- "학습 시작" 버튼 클릭
- AI가 문제를 생성할 때까지 대기 (약 10~30초)
- 문제를 풀고 즉시 해설 확인

### 4. 통계 확인
- 메인 화면에서 "학습 통계 보기" 클릭
- 총 학습 횟수, 정답률 등 확인

### 5. 취약점 분석
- 메인 화면에서 "취약점 분석" 클릭
- 자주 틀리는 주제, 오답 노트, 학습 추천 확인

## 프로젝트 구조

```
01. Accounting/
├── PDF(ocr)/                      # OCR 변환된 PDF 파일 저장 폴더
│   └── 02.계정과목별 정리_01.유동(당좌자산)-현금및현금성자산 회계처리.pdf
├── quiz_program/                  # 메인 프로그램
│   ├── gui_app.py                # GUI 메인 프로그램
│   ├── pdf_handler.py            # PDF 텍스트 추출
│   ├── quiz_engine.py            # AI 문제 생성 엔진
│   └── weakness_analyzer.py      # 취약점 분석 모듈
├── config.json                    # API 키 등 설정 (자동 생성)
├── learning_history.json          # 학습 기록 (자동 생성)
├── requirements.txt               # 필요한 패키지 목록
└── README.md                      # 사용 설명서
```

## 데이터 저장

### config.json
- API 키 저장
- 자동으로 생성되며 수동 편집 가능

### learning_history.json
- 모든 학습 세션 기록
- 문제, 답안, 정답률 등 저장
- 취약점 분석에 사용

## 문제 해결

### API 키 오류
- Google AI Studio에서 API 키가 유효한지 확인
- 무료 할당량을 초과했는지 확인

### PDF 읽기 오류
- PDF 파일이 OCR로 변환되어 있는지 확인
- 파일 경로에 한글이 포함되어도 괜찮습니다

### 문제 생성 실패
- 인터넷 연결 확인
- PDF 텍스트가 충분한지 확인 (최소 500자 이상 권장)
