# 📘 Notion 템플릿 커스터마이징 가이드

> 아미르님의 학습 관리 시스템을 Notion에서 완벽하게 구축하는 방법

---

## 🚀 빠른 시작 가이드 (5분 설정)

### 1단계: 기본 페이지 만들기

**① Notion 워크스페이스 생성**
```
1. Notion 접속 (notion.so)
2. 새 페이지 만들기 (+New Page)
3. 제목: "🎯 2026-2028 학습 관리"
4. 아이콘: 🎯 선택
```

**② 하위 페이지 구조 만들기**
```
🎯 2026-2028 학습 관리 (메인)
  ├─ 📊 대시보드
  ├─ 📅 주간 플래너
  ├─ 🤖 AI 프로젝트
  ├─ 📊 회계 학습
  ├─ 🗣️ 영어 학습
  └─ 📝 회고 노트
```

---

### 2단계: 템플릿 복사하기

**방법 1: 직접 복사 (추천)**
```
1. 제공된 .md 파일 열기
2. 내용 전체 복사 (Ctrl+A → Ctrl+C)
3. Notion 페이지에서 붙여넣기 (Ctrl+V)
4. Notion이 자동으로 마크다운 변환
```

**방법 2: 파일 Import**
```
1. Notion 사이드바 → Import
2. Markdown 선택
3. .md 파일 업로드
4. 자동으로 페이지 생성
```

---

## 🎨 데이터베이스 만들기

### AI 프로젝트 데이터베이스

**① 데이터베이스 생성**
```
1. /database 입력
2. "Table - Inline" 선택
3. 제목: "AI 프로젝트 목록"
```

**② 속성(Properties) 추가**
```
필수 속성:
- 프로젝트명 (Title) - 기본
- 상태 (Select): 계획중 / 진행중 / 완료
- 시작일 (Date)
- 완료일 (Date)
- 소요시간 (Number): 시간 단위
- 난이도 (Select): ⭐~⭐⭐⭐⭐⭐
- GitHub (URL)
- 기술스택 (Multi-select): Skills, n8n, Python 등
```

**③ 뷰(View) 설정**
```
뷰 1: 전체 목록 (Table)
- 모든 프로젝트 표시
- 상태별 정렬

뷰 2: 진행 중 (Board)
- 상태별로 칸반 보드
- 드래그 앤 드롭 가능

뷰 3: 완료 (Gallery)
- 완료된 프로젝트만
- 카드 형태로 보기
```

---

### 회계 학습 데이터베이스

**① 진도 관리 테이블**
```
속성:
- 장 (Title)
- 제목 (Text)
- 페이지 범위 (Text)
- 상태 (Checkbox)
- 완료일 (Date)
- 이해도 (Select): ⭐~⭐⭐⭐⭐⭐
```

**② 문제 풀이 기록**
```
속성:
- 날짜 (Date)
- 유형 (Select): 연습문제 / 기출문제 / 모의고사
- 범위/회차 (Text)
- 점수 (Number)
- 틀린 문제 (Text)
- 복습 완료 (Checkbox)
```

---

### 영어 학습 데이터베이스

**① 표현 노트**
```
속성:
- 표현 (Title)
- 의미 (Text)
- 카테고리 (Select): 이메일 / 회의 / AI/Tech
- 예문 (Text)
- 사용 완료 (Checkbox)
- 마지막 사용 (Date)
```

**② 독서 기록**
```
속성:
- 책 제목 (Title)

- 난이도 (Select)

```

## 🎯 대시보드 커스터마이징

### 진행률 바 만들기

**방법 1: Progress Bar (간단)**
```
1. /progress 입력
2. 자동 진행률 바 생성
3. 숫자 입력으로 퍼센트 표시
```

**방법 2: Formula (자동 계산)**
```
데이터베이스에서:
1. 새 속성 추가 → Formula 선택
2. 공식 입력:
   
   prop("완료시간") / prop("목표시간") * 100
   
3. 자동으로 달성률 계산
```

## 📊 자동화 설정

### 1. 버튼 만들기

**"이번 주 시작" 버튼**
```
1. /button 입력
2. 버튼 이름: "새로운 주 시작"
3. 액션 선택: "Add pages to database"
4. 주간 플래너 템플릿 선택
5. 클릭하면 자동으로 새 주간 페이지 생성
```

**"일일 체크" 버튼**
```
1. /button 입력
2. 버튼 이름: "오늘 할 일 체크"
3. 액션: "Open page"
4. 오늘 날짜 페이지로 이동
```

---

### 2. 알림 설정

**시험 D-Day 알림**
```
1. 회계 학습 페이지에서
2. 시험 날짜에 @remind 입력
3. 예: @remind 2026-06-01 (시험 2주 전)
4. Notion이 자동으로 알림
```

**주간 회고 알림**
```
매주 일요일 저녁 알림:
1. 주간 플래너에서
2. @remind every Sunday at 8pm
3. 자동으로 주간 회고 알림
```

---

### 3. 데이터베이스 연동

**대시보드에 여러 DB 표시**
```
1. 대시보드 페이지에서
2. /linked 입력
3. "Create linked database"
4. AI 프로젝트 DB 선택
5. 뷰를 "진행 중"으로 필터
6. 대시보드에 현재 진행 중인 프로젝트만 표시
```

## 🔧 고급 기능

### 1. Rollup으로 통계 계산

**전체 프로젝트 소요 시간 자동 계산**
```
1. 대시보드에 새 데이터베이스
2. Relation으로 AI 프로젝트 DB 연결
3. Rollup 속성 추가:
   - Relation: AI 프로젝트
   - Property: 소요시간
   - Calculate: Sum
4. 자동으로 총 시간 계산
```

---

### 2. Formula로 자동 계산

**주간 달성률 자동 계산**
```
공식 예시:
if(prop("실제시간") >= prop("목표시간"), 
   "✅ 목표 달성!", 
   "⚠️ " + 
   format((prop("목표시간") - prop("실제시간"))) + 
   "시간 부족")
```

## 🆘 문제 해결

### Q: 템플릿이 제대로 안 보여요
**A**: 마크다운 형식 확인
- Notion은 일부 마크다운만 지원
- 표가 안 보이면 /table로 다시 생성

### Q: 자동 계산이 안 돼요
**A**: Formula 문법 확인
- prop("속성명") 정확히 입력
- 속성 이름에 공백 있으면 안 됨

### Q: 모바일에서 느려요
**A**: 페이지 나누기
- 한 페이지에 너무 많은 DB는 무거움
- 연도별, 분기별로 페이지 분리
