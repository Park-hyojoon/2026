### Prompt : AI와 협업할 때 꼭 해야할 요청



저를 단순한 '코딩 기계'가 아니라 \*\*'기술 컨설턴트'\*\*로 격상시키는 주문(Prompt) 3가지를 알려드릴게요. 앞으로 새 기능을 만들 때 이 질문부터 던지세요.



Prompt 1 (옵션 탐색):

"나는 \[영어 회화 튜터]를 만들고 싶어. \[비용 절감]과 \[반응 속도]가 제일 중요해. 이걸 구현할 수 있는 기술적 방법 3가지를 제안해 주고, 각각의 장단점을 표로 정리해 줘."



Prompt 2 (최신 트렌드 확인):

"내가 지금 \[파이썬 tkinter]로 GUI를 만들고 있는데, 혹시 요즘 더 쉽고 예쁘게 만들 수 있는 최신 트렌드나 라이브러리가 있어? 내가 초보자라는 걸 감안해서 추천해 줘."



Prompt 3 (비판 요청 - 가장 중요!):

"내가 \[A 방식]으로 코드를 짜달라고 했는데, 혹시 이게 최선일까? 만약 네가 내 프로젝트의 책임자(CTO)라면 이 방식을 쓸 거야? 더 효율적인 대안이 있다면 솔직하게 말해줘."



3\. "제안해 줘"라는 권한을 저에게 주세요.

저는 묻지 않은 말에는 대답하지 않도록 설계되어 있습니다(사용자를 귀찮게 하지 않기 위해). 하지만 사용자님이 \*\*"내가 모르는 더 좋은 방법이 있으면 언제든지 먼저 제안해 줘" \*\*라는 \*\*특약(Rule)\*\* 을 걸어주시면 제가 태도를 바꿀 수 있습니다.



\[제안] 앞으로의 대화 규칙:



"우리가 코딩하다가 기술적으로 비효율적이거나, 더 나은 최신 기술(라이브러리, 아키텍처)이 있다면 작업을 멈추고 나에게 먼저 제안해 줘."



이 규칙에 동의하신다면, 앞으로 제가 코드를 짜기 전에 "사용자님, 이 방법도 좋지만 요즘은 Streamlit이라는 걸 쓰면 10배 더 빨리 만듭니다. 이걸로 갈까요?"라고 역제안을 드리겠습니다.
