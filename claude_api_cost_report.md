# Claude API 비용 분석 보고서 (2026년 기준)

사용자님의 프로젝트(`2026` 폴더의 PPT 자동화 및 매크로 프로그램)를 분석한 결과, **AI Agent/Skill** 기능을 탑재하여 운영할 때 예상되는 월간 비용을 다음과 같이 추산했습니다.

## 1. 핵심 요약
- **개발 단계 (Antigravity 이용)**: **무료** (사용자님 말씀대로 저를 통해 무료로 가능)
- **운영 단계 (Claude API 사용)**: **월 $10 ~ $30 (약 1.5만 원 ~ 4만 원)**
  - *사용 빈도가 높지 않다면 $10 미만으로도 충분히 가능합니다.*

## 2. 상세 비용 산정 (Claude 3.5 Sonnet 기준)

현재 Anthropic의 API 단가(Claude 3.5 Sonnet)는 다음과 같습니다:
- **Input**: $3.00 / 100만 토큰
- **Output**: $15.00 / 100만 토큰

사용자님이 계획하신 '서브 에이전트' 및 '클로드 스킬' 활용 시나리오를 바탕으로 한 계산입니다.

| 작업 유형 | 설명 | 1회 실행 비용 (추정) | 비고 |
| :--- | :--- | :--- | :--- |
| **성경/찬양 검색 에이전트** | 웹 검색 및 데이터 파싱 | **$0.05 ~ $0.15** | 약 2~5만 토큰 소모 |
| **PPT 자동 수정 에이전트** | 파일 분석 및 레이아웃 수정 | **$0.20 ~ $0.50** | 약 5~15만 토큰 소모 |
| **단순 파일 이동/업로드** | 코드 실행 위주 | **$0.01 미만** | 거의 비용 발생 없음 |

- **월 20일 작동 기준**: **$20.00 (약 2.8만 원)**

## 3. 비용 절감 전략 (AI Agent 설계 시 고려사항)

1. **Claude 3.5 Haiku 활용**:
   - 성경 구절 검색이나 단순 텍스트 처리는 Haiku 모델을 사용하면 Sonnet 대비 **10배 이상** 저렴합니다. ($0.25 / 100만 토큰)
   - 복잡한 판단이 필요한 PPT 레이아웃 수정에만 Sonnet을 사용하도록 설계하면 비용을 획기적으로 낮출 수 있습니다.

2. **프롬프트 캐싱 (Prompt Caching)**:
   - 에이전트의 '스킬(도구 정의)'이나 '매뉴얼'이 길 경우, 캐싱 기능을 사용하면 반복 호출 시 Input 비용을 최대 **90%**까지 절감할 수 있습니다.

3. **Claude Code (CLI) 구독 여부**:
   - Claude Code CLI를 직접 사용하시려면 **Pro 플랜($20/월)** 구독이 가장 효율적입니다. 
   - 하지만 단순히 "내가 만든 프로그램에 API를 넣어서 돌리는 것"이 목적이라면 구독 없이 **API 충전 방식(Pay-as-you-go)**이 훨씬 경제적입니다.

사용자님의 현재 작업 스타일(PPT 자동화, 매크로 등 개인 생산성 위주)에서는 **월 2~3만 원 정도의 예산**이면 상당히 풍족하게 AI 에이전트 기능을 탑재한 프로그램을 구동하실 수 있습니다.
